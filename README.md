# PackersTools

![License](https://img.shields.io/badge/license-PSAL-blue.svg)
[![Donate](https://img.shields.io/badge/Donate-PayPal-green.svg)](https://paypal.me/Oggals)

PackersTools is a simple, fast, and reliable tool for creating Microsoft Intune Win32 packages using the official Microsoft Win32 Content Prep Tool.

It provides automation, installer integration, and workflow improvements for IT professionals managing application deployment in Microsoft Intune.

---

## Features

* Easy packaging of Win32 applications
* Uses official Microsoft packaging tool
* Simple installer deployment
* Lightweight and fast
* Designed for IT professionals and consultants
* No external dependencies required

---

## Installation

Download the latest installer from:

https://github.com/OscarFreij/PackersTools/releases

Run the installer and follow the instructions.

---

## Usage

After installation, PackersTools is available from:

Start Menu or installation directory.

Follow the included instructions to package your application.

---

## Requirements

* Windows 10 or later
* Microsoft Intune environment
* Administrator privileges (recommended)
