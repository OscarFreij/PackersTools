# PackersTools

## What it does
- Adds "Right Click Tools" for application packers
- Designed for PSADT v3 and v4 but supports others

## Installation
- Download installer from Releases
- Run as administrator

## Silent Install
installer.exe /VERYSILENT /SUPPRESSMSGBOXES /NORESTART

## Silent Uninstall
"C:\Program Files\PackersTools\unins000.exe" /VERYSILENT

## Security Notice
This installer is currently unsigned.
Windows may show a SmartScreen warning.
The source code is fully available in this repository.
